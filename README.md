# VeruClock ⏰

A precise web-based clock designed for advanced customizations options.

VeruClock is a responsive web application that displays accurate synced time for the specified timezone or local time if user decides, designed with a clean UI and smooth interactions. This project focuses on real-time data updating, and offers more time features that you can find in modern apps online.

---

## 🚀 Features

* **Real-time Accuracy**
* **Custom timezone sync**
* **Responsive Design** — Under development.
* **Timer** — Under development.
* **StopWatch** — Under development.
* **Light and Dark Theme Colors** — Under development.

## 💻 How to Run Locally

1. **Clone the repository**
   
   ```bash
   git clone https://github.com/Dawud-Shuaibu/veruclock.git
   ```
   
3. **Navigate to the project folder**

   ```bash
   cd veruclock
   ```
   
5. **Run the typescript compiler (Requires installed *Node.js* and Typescript)**

   ```bash
    tsc
    ```
   
    Click [here](https://nodejs.org/en/download) to download and install *node.js*, then open your terminal and run:

   ```bash
   npm install -g typescript
   ```
   
   Verify:

   ```bash
   tsc -v
   ```
   
7. **Open the App**
   
   Simply open `index.html` in your desired web browser and ***Explore***
